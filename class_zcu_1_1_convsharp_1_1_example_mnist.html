Detailed Description

Example of simple CNN which will classify MNIST dataset We will use convolution layer followed by ReLU activation and maxpooling layer. After that we will use flatten layer and linear layer with 64 neurons and ReLU activation. In the output layer is placed 10 neurons with softmax function. Compressed dataset: http://yann.lecun.com/exdb/mnist/
